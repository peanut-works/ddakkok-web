## 화면 1: 아동 관리 (Children Management)

### 개요
앱에서 담임 교사가 등록한 아동 프로필을 원장·관리자가 전체 조회·편집·관리하는 화면.
알레르기, 피부 민감도, 건강 특이사항을 한눈에 파악하고 필터·검색 가능.

### 레이아웃 구조
```
[상단 헤더] 아동 관리 타이틀 + [+ 아동 등록] 버튼(파랑 filled)

[필터 바]
  반 선택: 전체 | 햇님반 | 별님반
  알레르기 필터: 전체 | 우유 | 계란 | 땅콩 | 밀 | 기타
  검색: "아동 이름 검색"
  우측: 총 28명

[아동 카드 그리드] — 4열 카드 배열, 각 카드:
  ┌──────────────────────┐
  │ [아바타 48px] 이름 나이│
  │ 햇님반 · 담임: 김선생 │
  │ ─────────────────── │
  │ 알레르기 태그들        │
  │ [우유] [계란] [땅콩]  │
  │ ─────────────────── │
  │ 피부: 아토피 주의      │
  │ 특이사항: 없음        │
  │ ─────────────────── │
  │ 마지막 검사: 05.16    │
  │          [상세 보기>] │
  └──────────────────────┘

알레르기 태그: rounded-full, 빨간 배경(#FFF0F0), 빨간 텍스트(#FF6B6B), 11px Bold
피부 민감 태그: rounded-full, 노랑 배경(#FFF8CC), 노랑 텍스트(#C9A800)
```

### 아동 상세 슬라이드 패널 (우측에서 슬라이드인)
```
[아동 프로필 헤더]
  아바타 64px + 이름(22px Bold) + 나이 + 반
  담임 교사: 김관리 / 마지막 수정: 2025.05.16

[탭] 기본 정보 | 건강 프로필 | 제품 검사 이력 | 보호자 연락처

[건강 프로필 탭]
  알레르기 목록 (추가/삭제 가능)
  피부 상태: 정상 / 민감 / 아토피
  기타 특이사항 텍스트

[제품 검사 이력 탭]
  최근 검사한 제품 리스트 (상태 배지 + 제품명 + 날짜)
```

### 샘플 데이터
```
아동 8명 (id 1~8):
  김서준(5세, 햇님반) — 알레르기: 없음, 피부: 정상
  이지은(4세, 햇님반) — 알레르기: 우유·계란, 피부: 민감
  박민준(5세, 별님반) — 알레르기: 땅콩, 피부: 정상
  최서연(3세, 별님반) — 알레르기: 우유, 피부: 아토피
  정도윤(4세, 햇님반) — 알레르기: 없음, 피부: 정상
  이유나(4세, 별님반) — 알레르기: 계란·밀, 피부: 민감
  강하준(5세, 햇님반) — 알레르기: 향료, 피부: 정상
  조하나(4세, 별님반) — 알레르기: 우유·땅콩, 피부: 아토피
```

## 화면 2: 제품 분석 (Product Analysis)

### 개요
담임 교사가 앱으로 스캔한 모든 제품의 분석 결과를 관리자가 조회·관리하는 화면.
제품별 안전성 현황, 재고 등록, 유효기간 관리, 신규 제품 등록 기능 포함.

### 레이아웃 구조
```
[상단 헤더] 제품 분석 타이틀
  우측 버튼: [바코드 스캔] [+ 제품 등록] (파랑 filled)

[요약 카드 3개] — 가로 배열
  전체 분석 제품: 62개
  주의·보류 제품: 9개 (노랑 경고 아이콘)
  유효기간 임박: 3개 (주황 시계 아이콘)

[탭 바]
  전체 | 안전 | 주의 필요 | 사용 보류 | 유효기간 만료 | 확인 필요

[제품 테이블/리스트 — 2가지 뷰 전환: 테이블뷰 / 카드뷰]

테이블뷰:
  ┌──────┬────────────────────────┬───────┬────────────┬───────────┬──────────┐
  │체크박스│ 제품명                  │ 상태  │ 바코드       │ 유효기간   │ 마지막분석│
  ├──────┼────────────────────────┼───────┼────────────┼───────────┼──────────┤
  │  □   │ 퓨어 물티슈 캡형 (100매) │[안전] │880000001012│2026.11.20 │ 05.16    │
  │  □   │ 키즈 샴푸 (500ml)        │[주의] │880000001023│2025.08.15 │ 05.16    │
  │  □   │ 베이비 로션 (300ml)      │[안전] │880000001034│2025.05.31 │ 05.16    │
  └──────┴────────────────────────┴───────┴────────────┴───────────┴──────────┘
  
  선택 시 하단 액션바: [일괄 보류 처리] [일괄 삭제]

카드뷰:
  제품 썸네일 + 이름 + 상태 배지 + 유효기간 D-day 표시
```

### 제품 상세 슬라이드 패널
```
[제품 헤더]
  썸네일 80px + 제품명(Bold) + 제조사 + 바코드
  상태 배지 (크게)

[탭] 성분 분석 | 아동별 안전 판정 | 검사 이력

[아동별 안전 판정 탭] ← 핵심 기능
  해당 제품을 사용한 경우 아동별로 PASS/WARN/FAIL 결과를 나열
  예:
    김서준 (5세) ────────── [안전]
    이지은 (4세) ─ 우유 알레르기 [주의]
    조하나 (4세) ─ MIT 성분 포함 [보류]

[성분 분석 탭]
  주요 성분 목록
  위험 성분 강조 표시 (빨간 하이라이트)
```

## 화면 3: 안전 결과 (Safety Results)

### 개요
전체 아동 × 전체 제품의 안전 판정 매트릭스를 관리자가 한눈에 볼 수 있는 화면.
기간별·반별·상태별 필터링, 위험 아동/제품 우선 표시.

### 레이아웃 구조
```
[상단 필터 바]
  기간: 오늘 | 이번주 | 이번달 | 직접 입력
  반: 전체 | 햇님반 | 별님반
  상태: 전체 | 주의 | 보류
  제품: "제품명 검색"
  우측: [엑셀 내보내기] [PDF 리포트]

[요약 섹션 — 상단 4개 수치 카드]
  총 검사 건수: 126건
  안전: 84건 (66.7%)
  주의 필요: 30건 (23.8%)
  사용 보류: 12건 (9.5%)

[메인 콘텐츠 2컬럼]

  [좌 — 아동 × 제품 히트맵 매트릭스]
  행: 아동 이름 / 열: 제품명
  셀: 상태 컬러 점 (●안전 ▲주의 ✕보류 회색-미검사)
  스크롤 가능, 셀 클릭 시 상세 툴팁

  예시:
            │ 물티슈 │ 샴푸 │ 로션 │ 선크림│
  ──────────┼───────┼─────┼──────┼───────┤
  김서준(5세)│  ●   │  ●  │  ●  │   ●  │
  이지은(4세)│  ▲   │  ●  │  ▲  │   ●  │
  조하나(4세)│  ✕   │  ▲  │  ✕  │   ▲  │

  [우 — 위험 알림 리스트]
  "즉시 조치 필요" 섹션:
    조하나 + 세탁세제 → FAIL (MIT 성분)
    이유나 + 로션 → WARN (계란 성분)
  
  "최근 7일 주요 이슈" 타임라인:
    05.16 베이비 로션 — 이지은 주의 판정
    05.15 키즈 샴푸 — 강하준 주의 판정

[하단 — 전체 판정 기록 테이블]
  날짜 | 아동 | 제품 | 담임 | 상태 | 상세보기
  정렬, 필터, 검색 가능
```

## 화면 4: 보호자 공유 (Parent Sharing)

### 개요
담임 교사가 앱으로 공유한 안전 결과를 원장이 확인하고, 미공유 건을 관리하는 화면.
보호자별 수신 현황, 공유 이력, 일괄 공유 기능.

### 레이아웃 구조
```
[상단 헤더]
  "보호자 공유 관리" 타이틀
  우측: [미공유 건 일괄 공유] 버튼 (파랑 filled) + 미공유 수 배지

[상태 요약 카드 3개]
  공유 완료: 24건
  미공유 (주의 이상): 4건 → 빨간 강조
  오늘 공유 예정: 7건

[탭] 미공유 | 공유 완료 | 전체 이력

[미공유 탭 — 우선순위 리스트]
  ┌──────────────────────────────────────────────────┐
  │ ● FAIL  조하나 부모님           D+0  10분 전 등록 │
  │  퓨어 물티슈 — MIT 성분 포함                      │
  │  담임: 김관리 선생님                              │
  │                    [공유 메시지 미리보기] [공유하기]│
  └──────────────────────────────────────────────────┘
  (우선순위: FAIL > WARN 순 정렬)

[공유 완료 탭]
  보호자명 | 제품 | 상태 | 공유 일시 | 수신 확인 여부
  수신 확인: ✓ 읽음 / 미읽음 표시

[공유 메시지 미리보기 모달]
  받는 분: 조하나 보호자님
  내용: "안녕하세요. [퓨어 물티슈] 제품 검사 결과를 안내드립니다.
        조하나 어린이 기준 [사용 보류] 판정이 나왔습니다.
        사유: MIT 성분 포함..."
  하단: [수정] [공유하기] 버튼
```

### 우측 사이드 위젯
```
[보호자 연락처 퀵 패널]
  자주 공유하는 보호자 목록
  클릭 시 해당 보호자 공유 이력 바로 표시

[공유 통계]
  이번달 공유 건수: 47건
  평균 응답 시간: 2.3시간
  미확인율: 12%
```

## 화면 5-B: 식단 관리 (Meal Planning) — 영양사 핵심 화면

### 개요
아동 알레르기·건강 프로필(앱 데이터 연동)을 기반으로 주간 식단표를 관리하고,
식재료·간식이 특정 아동에게 안전한지 사전 체크하는 화면.

### 레이아웃 구조
```
[상단 헤더]
  "식단 관리" 타이틀
  우측: 주간 선택 (◀ 2025년 5월 3주차 ▶) + [식단표 저장] [PDF 출력]

[2컬럼 레이아웃]
  좌측 70%: 주간 식단 캘린더
  우측 30%: 알레르기 현황 패널

────────────────────────────────
[좌측 — 주간 식단 캘린더]
────────────────────────────────
요일별 5컬럼 테이블:

       │  월(19)  │  화(20)  │  수(21)  │  목(22)  │  금(23)  │
───────┼──────────┼──────────┼──────────┼──────────┼──────────┤
아침   │ 흰쌀밥   │ 잡곡밥   │ 흰쌀밥   │ 잡곡밥   │ 흰쌀밥   │
       │ 미역국   │ 된장국   │ 콩나물국 │ 김치국   │ 무국     │
───────┼──────────┼──────────┼──────────┼──────────┼──────────┤
점심   │ 소불고기 │ 닭볶음탕 │ 제육볶음 │ 생선구이 │ 돈가스   │
       │ 시금치나물│ 감자조림 │ 두부조림 │ 깍두기   │ 샐러드   │
───────┼──────────┼──────────┼──────────┼──────────┼──────────┤
```
